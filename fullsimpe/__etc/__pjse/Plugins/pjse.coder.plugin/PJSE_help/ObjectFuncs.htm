The Object Function Plugin (OBJf)

 

Certain events happen in the Sims world that require objects to act or Sims to interact with them without a menu option being selected.  These events have placeholders in the Object Function table, and if you would like your object to carry out a reaction or interaction when one of these events occur, then you record the BHAV you would like to run for this event in the OBJf table.

 

Headers

These values refer to the object function file as a whole

Filename Any descriptive title you like
Commit Updates the package file in memory to include edits to current resource.  No change made to file on disk.  If you don't commit, you can abandon edits to current resource by simply opening another resource.

.

Input boxes

These values refer to the currently selected event.  The button to the right of each box opens up a picker of all available functions to choose from.

Action BHAV This is the code that will be run when the event happens
Guardian BHAV This is the optional BHAV containing any test conditions that must be met in order for the action to run.
