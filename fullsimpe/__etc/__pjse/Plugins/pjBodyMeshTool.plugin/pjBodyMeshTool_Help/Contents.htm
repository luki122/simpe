PJ Body Mesh Tool Help

 

This tool is to help with the previously very tedious parts of extracting and linking a mesh ready for making a new clothing, hair, or accessory mesh for your Sims.  Tutorials to help with the task overall may be found in many places, including Mod The Sims 2

 

Installing

If your installation was not already part of SimPE, or you are picking up a new version from the Simlogical website then install as follows:

  1. Unpack the zip and drop the enclosed "Plugins" folder into your SimPE program folder.   This should mix the new plugins in with any you already have in your existing plugins folder there.   That is the correct result.   People who have downloaded earlier version of this will be asked if it is ok to overwrite existing files?  Yes it is.

 

Extracting mesh parts

This takes care of finding and extracting and importing those four mesh files from the Sims0x.package files, in a fraction of the time it used to take.  This assumes you have already made the outfit (or hair etc) in Bodyshop cloned from the outfit you want to base your new mesh on.

  1. Open SimPE, and start a new file, using File|New

  2. Use menu option Tools|PJSE|Body Mesh Tool|Extracting stage, and browse to the outfit package you already made in Bodyshop that you wanted to base your new mesh on.

  3. You should soon see the 4 files appearing (or more if you had more than one Propertyset or XMOL in your Bodyshop file)  NB the new file has not been saved at this point.

  4. You are now ready to continue the normal process from the bit where you have to Fix Integrity.

 

Linking mesh parts to Bodyshop file

This part is instead of where you would extract the Shape and Resource node resources, add them to the Bodyshop package, add them to the 3IDR list then delete them again.

  1. Open your Bodyshop file

  2. Open the relevant 3IDR resource (there may be several for accessories)

  3. Use menu option Tools|PJSE|Body Mesh Tool|Linking stage, ensuring you didn't have any edits waiting to be saved, as they may be lost after linking.   Browse to your mesh file and click OK

  4. After the tool tells you it completed, close your 3IDR resource without Committing by selecting another resource to open.   The linking should now be complete, and you should save your file.  If you wish, you may reopen the 3IDR first to satisfy yourself the right changes were made.
