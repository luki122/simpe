Behaviour Functions Overview

 

Behaviour functions (BHAVs) contain the code that controls in a detailed way what will happen when a Sim goes about his life.  They cover everything from how he handles the current object to what happens when he starves to death.  From NightLife onwards, some of the behaviour is scripted in LUA scripts, which can be called from BHAVs.

To make your work easier, you may choose to provide labels for the Local Variables and Parameters declared in a BHAV by setting up an optional corresponding TPRP resource.  Maxis did not include these with their distributed packages, probably to save file size.
