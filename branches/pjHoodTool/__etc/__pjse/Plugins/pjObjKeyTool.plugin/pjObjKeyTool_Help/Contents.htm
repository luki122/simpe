PJ ObjKey Tool Help

This tool is designed to gather all the resources for an object into a single package to make it easier to customise. It's based on wes_h's tutorial on making custom collars and uses the same terms as that, i.e. ObjKey, FragKey, BinKey, etc.

You have to decide on an object to work on from Skins.package. Other .package files may or may not work...

Extract the overlay (GZPS, XMOL, XTOL) or matching index (3IDR) into a new .package. For example, ducollarsmooth_bonesyellow XMOL or 3IDR. Open the new package and select the resource you put in it.

Then you click on Tools, PJSE, ObjKey Tool and wait whilst it hunts for

  1. the matching other half of the ObjKey
  2. all the FragKeys that reference the ObjKey
  3. the BinKeys for those FragKeys
  4. the string resources for the BinKeys and FragKeys
  5. the rcol chain hanging off the ObjKey, including TXMTs, TXTRs and LIFOs
These are all imported into the open package.

Then it's back to you to customise the object.
