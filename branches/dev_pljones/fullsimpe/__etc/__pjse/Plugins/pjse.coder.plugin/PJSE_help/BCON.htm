The Behaviour Constants Editor Plugin (BCON)

 

The BCON file is a very simple file that just holds a series of numerical values.  The idea of it is that you can just edit a value in one place when you are tuning levels for your BHAVs to use.   If you wish to have labels to remind you what the values represent, you can create some in a TRCN file with the same instance number

 

Headers

These values refer to the BCON file as a whole

Filename Any descriptive title you like
Flag No idea!
Commit Updates the package file in memory to include edits to current resource.  No change made to file on disk.  If you don't commit, you can abandon edits to current resource by simply opening another resource.

.

Value input boxes

Hex Just two alternative modes of input
Dec What you type into one will be displayed in the other in the alternative notation

 

Buttons

Add Value Adds entry to end
Delete Value Deletes selected entry.  Remaining entries will be renumbered to close gap
Cancel Undoes last edit.
Up arrow Selects earlier line
Down arrow Selects later line
Make labels Creates a TRCN resource with a dummy label (ready for editing) for each of the BCON entries you have.  Gives it the same Instance number as your current BCON.   You can use this button again if you add BCON entries, and it will then overwrite the current values (Keep existing labels? No) or append lines for the additional ones (Keep existing labels? Yes)

 

Keyboard Shortcuts

Alt-A Add value
Alt-L Delete value
Alt-D Down (Next value)
Alt-U Up (Previous value)
