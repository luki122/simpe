The Constant Labels Editor Plugin (TRCN)

 

We don't have many examples of original Maxis TRCN resources.   There is a Wiki description of the fields, but that does not appear to apply to those examples we do have available, by MaxoidMonkey in his downloadable objects.   The PJSE TRCN editor has been designed to correlate to the fields as used by MaxoidMonkey.   This means line 0x0 will be used as the label for BCON 0x0 of the corresponding instance number, and so on.   The fields that PJSE does not use, remain however visible and editable, in case you ever want to use a tool that uses these labels according to the Wiki.

 

Headers

These values refer to the TRCN file as a whole

Filename Any descriptive title you like
Flag No idea!
Commit File Updates the package file in memory to include edits to current resource.  No change made to file on disk.  If you don't commit, you can abandon edits to current resource by simply opening another resource.

.

Value input boxes

Label Type a string in here, as you wish the label to appear when you are editing the BCON or BHAV that uses it.
ID, Used, DefValue, MinValue, MaxValue These are not used by PJSE.  If you are going to use a tool which does use them, please refer to that tool's documentation.

 

Buttons

Add Label Adds entry to end
Delete Label Deletes selected label.  Remaining entries will be renumbered to close gap
Cancel Undoes last edit.
Up arrow Selects earlier line
Down arrow Selects later line

 

Keyboard Shortcuts

Alt-A Add label
Alt-L Delete label
Alt-D Down (Next label)
Alt-U Up (Previous label)
