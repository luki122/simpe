Features available from anywhere

 

From within SimPE at any time:

From the application Menu Bar                     

Tools

 
    | PJSE

 
        | Refresh File Table

Rereads files from disk.   Useful for instance if you are updating files in two instances of SimPE at the same time and wish PJSE to reread the information from them.  PJSE  keeps a filetable cache separately from the SimPE application as a whole, so refreshing one does not refresh the other.   The PJSE filetable affects the interpreting of things like BHAV names and strings for display in the PJSE editors.   Read more about the PJSE filetable here

 
        | Resource Finder

Brings up a dialog where you type in either the GUID or part of the resource name of the object or resource you want to find and it lists any matches found within files in the PJSE filetable.  You can copy text from the results to paste into other input boxes such as the SimPE group filter.  Read more about the Resource Finder here

            | PJSE editors

 
                  

Choose from known attributes

This is used in the BHAV wizards, so if there is an expression referring to an attribute of the object, the dropdown picker will show the attribute name instead of its number - for those attributes that are labelled.   This can also be set in the wizard itself and the last choice will be remembered.

 
                   

Display special buttons panel

An alternative place to set an option that is also in the BHAV editor

 
                   

Reveal default language

An alternative place to set an option that is also in the strings editor

 
                   

Reveal string descriptions

An alternative place to set an option that is also in the strings editor

 
 

Load default GUID index at startup

Only useful if you edit BHAVs.  More information in BHAV editor

 
            | PJSE Update Tool

 
 

Allow PJSE to check for updates

If Daily, PJSE will check online once per calendar day, on days that you open a PJSE editor.  "Ask me" means it will come up and ask you if you want it Daily or Manual, next time you start SimPE.   "Ask me" doesn't stay that way after asking you, it was mainly for test purposes.

 
 

Update URL

Normally you should not change this.  It is available in case anything happens to the site that hosts PJSE and we have to move its location.   Then we would let you know by other means and you can change this manually.

 
            | PJSE FileTable

 
 

Load the PJSE FileTable at startup

Choose whether to load it at startup of SimPE or only when you first go into a PJSE plugin.  You won't want this loading at startup if most of your work is with non-PJSE plugins such as recolouring, because it will add to startup time for no benefit.
