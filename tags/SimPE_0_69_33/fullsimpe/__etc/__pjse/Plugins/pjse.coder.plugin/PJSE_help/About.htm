About PJSE

PJSE stands for Peter Jones' Simantics Editor.  It is a collection of plugins for SimPE which provide enhanced features above those available in the built-in editors.  When you install SimPE, it is set to install PJSE plugins by default, and those will be the ones you automatically get while using SimPE unless you disable them or untick the option during installation.

Please remember SimPE itself is still in Alpha stage, which means these plugins should be regarded as Alpha too.  For those who don't know what that means, it means unless you consider yourself to be technically literate, or happy to undertake adventures, you should not use it.

The editors covered by PJSE currently are:

  • BCON - Behaviour constants
  • TRCN - Behaviour constant labels
  • BHAV - Behaviour functions
  • TPRP - Behaviour function labels for Parameters passed in and Local Variables
  • STR# - Textlist files
  • CTSS - Build/Buy catalog object name and description
  • TTAB - Pie Menu function list and attraction settings
  • TTAs - Pie Menu labels
  • OBJf - Object Functions table

 

PJSE comes with credit to Quaxi for SimPE, Dizzy2 and Shy and probably others for the DisaSim parsing details, DatGen team for some data info, especially Breon for the Wiki entries, and everyone who has patiently helped with testing.

The source for the PJSE project can be accessed on SourceForge under the name of "plj-simpe"
