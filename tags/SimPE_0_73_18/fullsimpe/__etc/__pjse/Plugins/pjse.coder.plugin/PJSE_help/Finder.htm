The PJSE Resource Finder

 

This can be found under Tools | PJSE | Resource Finder

Unlike the Finder which is a tab with binocular icon, this Finder concentrates on resources of interest to code modders - ie Simantics resources.  Therefore it searches resources on the PJSE filetable rather than the overall SimPE filetable.

 

Search for resources in... Entire PJSE filetable Objects.package plus any paths in packages.txt
  Objects.package only This refers to your live objects.package - the one known to represent your latest EP
  Current package only Particularly useful if you are looking for the resource in an earlier objects.package.  Open the package then use this option.

 

 

Search resource for... GUID Requires you to fill in number in GUID box below.  "0x" may be left off.
  Any other Requires string in "Name" box below.  Should be substring of resource name. Do not use wildcard symbols.
You can tick any combination of boxes in this group.  Search will match any on an "OR" basis.

 

Click Search.

List of found resources along with their paths will build up.  Search can be stopped at any time. Copy from search window is possible, and this can be useful to copy a group number to use in SimPE's filter groups option.  Populated search window may be left open for continued reference.
