The Behaviour Labels Editor Plugin (TPRP)

 

The TPRP resource holds labels for the Local Variables and Parameters used in the BHAV resource with the corresponding instance number.

 

Headers

These values refer to the TPRP file as a whole

Filename Any descriptive title you like.  It is probably helpful to call it the same as the BHAV it labels.
Version Not sure the significance.  I have never felt like changing the default value.
Commit File Updates the package file in memory to include edits to current resource.  No change made to file on disk.  If you don't commit, you can abandon edits to current resource by simply opening another resource.

.

Value input boxes

Label Type a string in here, as you wish the label to appear when you are editing the BHAV that uses it.

 

Buttons

Add Label Adds entry to end
Delete Label Deletes selected label.  Remaining entries will be renumbered to close gap
Cancel Undoes last edit.
Up arrow Selects earlier line
Down arrow Selects later line
Sideways arrows Switch between tabs for Parameters and Local Variables labels

 

Keyboard Shortcuts

Alt-A Add label
Alt-L Delete label
Alt-D Down (Next label)
Alt-U Up (Previous label)
Alt-> Next tab
Alt-< Previous tab
